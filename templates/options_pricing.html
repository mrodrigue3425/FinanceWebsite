Options Pricer: Select a Model

Welcome to the Options Pricer interface. Please use the **Valuation Methods** menu on the left to select a methodology to begin working with the tool.

Tip: Selecting a method will load its corresponding input form and calculation interface in this area.
  • The **Black-Scholes** model provides a closed-form analytical solution.
  • **Finite Difference Methods** use numerical techniques to solve the Black-Scholes PDE.
  • **Machine Learning** approaches leverage AI for fast, non-parametric pricing.
  • **Fourier Transforms** (e.g., Carr-Madan) are ideal for models with non-Gaussian characteristics.
